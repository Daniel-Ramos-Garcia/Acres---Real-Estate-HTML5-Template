Getting Started

Navigation

For now, we provide 1 header style only, which can be alternated between 3 states:

Sticky by adding class can-sticky to the header element

Absolute by adding class aboslute-header to the header element

Static default behaviour

Spacing

Aside from bootstrap spacing utility classes, we provide 2 spacing classes for section seperation

Default spacing by adding class section to any element

Irregular spacing by adding class section-padding to any element (This is used incase you have elements with bottom margins)

Google Fonts

By default, Acres uses Open Sans, and Poppins as the main fonts accross the whole template. You can change that by accessing the style.css file and changing the Font URL at the very start of the file. Simply go to Google fonts, select the font you like, and replace the link with the one we currently have in the style sheet.

Maps

Acres uses Mapbox as it's primary map library. To be able to start using the map, you must first grab an API from the mapbox Website.

Simply Sign in, and you will be redirected to your dashboard where your API key belongs. Copy that key and paste it in the files as described below.

All code for the maps lives in assets/js/map.js & submit-listing-map.js. Those files are linked under Page Specific Scripts & Styles before the end of the body tag and in the head. And in those files, is where you will be replacing your API Key
